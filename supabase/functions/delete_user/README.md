# Edge Function: delete_user

Umożliwia użytkownikowi usunięcie własnego konta z poziomu aplikacji.

## Wdrożenie

```bash
supabase functions deploy delete_user
```

**Brak dodatkowej konfiguracji** – zmienne `SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY` są automatycznie wstrzykiwane przez Supabase.

Szczegółowa instrukcja: [docs/INSTRUKCJA_EDGE_FUNCTIONS.md](../../../docs/INSTRUKCJA_EDGE_FUNCTIONS.md)
